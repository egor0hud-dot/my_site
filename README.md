# Веб-сайт на Flask

Полнофункциональный веб-сайт на Python Flask с формой обратной связи, кейсами и админ-панелью.

## 🚀 Возможности

- **Главная страница** с hero-блоком и описанием
- **5 кейсов** с детальными описаниями:
  - Разработка Telegram-бота для бизнеса
  - Создание интернет-магазина
  - Автоматизация CRM
  - Разработка корпоративного сайта
  - AI-ассистент для поддержки клиентов
- **Форма обратной связи** с валидацией и сохранением в БД
- **Админ-панель** с авторизацией для управления заявками
- **Современный дизайн** с Bootstrap 5 и кастомными стилями
- **Адаптивная верстка** для всех устройств
- **Анимации и hover-эффекты**

## 📋 Требования

- Python 3.8 или выше
- pip (менеджер пакетов Python)

## 🛠️ Установка

1. **Клонируйте репозиторий или скачайте проект**

2. **Создайте виртуальное окружение (рекомендуется):**

```bash
python -m venv venv
```

3. **Активируйте виртуальное окружение:**

   - Windows:
   ```bash
   venv\Scripts\activate
   ```

   - Linux/Mac:
   ```bash
   source venv/bin/activate
   ```

4. **Установите зависимости:**

```bash
pip install -r requirements.txt
```

## ▶️ Запуск

Запустите приложение одной командой:

```bash
python app.py
```

Приложение будет доступно по адресу: `http://localhost:5000`

## 🔐 Доступ к админ-панели

- URL: `http://localhost:5000/admin/login`
- Логин по умолчанию: `admin`
- Пароль по умолчанию: `admin`

**⚠️ ВАЖНО:** В продакшене обязательно измените учетные данные администратора через переменные окружения или в файле `config.py`.

## 🎨 Технологии

- **Backend:**
  - Flask 3.0.0
  - Flask-Login (авторизация)
  - Flask-WTF (формы и CSRF защита)
  - SQLAlchemy (работа с БД)
  - SQLite (база данных)

- **Frontend:**
  - Bootstrap 5.3.2
  - Bootstrap Icons
  - Кастомный CSS с анимациями
  - Vanilla JavaScript

## 📝 Основные функции

### Форма обратной связи

- Валидация всех полей на сервере и клиенте
- Сохранение заявок в базу данных
- Уведомления об успешной отправке
- Защита от CSRF атак

### Админ-панель

- Авторизация администратора
- Просмотр всех заявок
- Отметка заявок как "прочитано"
- Удаление заявок
- Статистика (всего заявок, непрочитанных, прочитанных)

### Кейсы

- 5 готовых кейсов с описаниями
- Детальные страницы для каждого кейса
- Красивые карточки с иконками
- Анимации при наведении

## 🔧 Настройка

### Изменение учетных данных администратора

1. **Через переменные окружения (рекомендуется):**

```bash
# Windows
set ADMIN_USERNAME=your_username
set ADMIN_PASSWORD=your_password

# Linux/Mac
export ADMIN_USERNAME=your_username
export ADMIN_PASSWORD=your_password
```

2. **Или измените в `config.py`:**

```python
ADMIN_USERNAME = 'your_username'
ADMIN_PASSWORD = 'your_password'
```

### Изменение секретного ключа

В продакшене обязательно измените `SECRET_KEY` в `config.py` или через переменную окружения:

```bash
set SECRET_KEY=your-secret-key-here
```

## 🐛 Решение проблем

### Ошибка при установке зависимостей

Убедитесь, что используете Python 3.8+:
```bash
python --version
```

### Ошибка подключения к базе данных

База данных создается автоматически при первом запуске. Если возникают проблемы, удалите файл `site.db` и запустите приложение снова.

### Порт 5000 занят

Измените порт в `app.py`:
```python
app.run(debug=True, host='0.0.0.0', port=5001)
```
